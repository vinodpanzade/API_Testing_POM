🚀 API Test Automation Framework (Cypress)
A scalable API Automation Testing Framework built using Cypress, BDD (Cucumber), and POM architecture to validate REST APIs with database verification and CI/CD integration.

The framework simulates real-world backend testing by combining API validation, database checks, dynamic data generation, and automated schema validation.

🛠 Tech Stack

Cypress

JavaScript

Cucumber (BDD)

MongoDB

Jenkins

Faker.js

⚡ Key Features

Automated CRUD API testing (GET, POST, PUT, DELETE)

Positive and negative test scenarios

API chaining for end-to-end workflow validation

Data-driven testing

Dynamic test data generation using Faker.js

MongoDB database validation to verify backend data

Schema contract testing for API response validation

Reusable Cypress custom commands

CI/CD execution using Jenkins
